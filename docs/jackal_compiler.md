# Jackal Compiler

Jackal compiler is part of the project to build the binary code
to be execute by __jackalvm__.

# Instructions

| Load the constant to the stack | JKL_CST                  |
|--------|--------------------------------------------------|
| opcode | 0x00fc                                           |
| nargs  | 0x02                                             |
| arg1   | Index of the const                               |
